D3 within React the right way

The problem

  • To use D3 with React, you typically need to pulls the SVG node out of the virtual DOM and manipulate the SVG by yourself. Thats what react-d3-wrap does.
  • d3-react inject a plugin into d3's prototype and convert the detached and mutated DOM node to React elements for you.
    • It limits the D3 API you can use in your components
    • It blows away all the virtual DOM goodness, throwing away the real DOM tree after every mutation and render!

Solution

  • Build a fake DOM that supports just enough methods to get the D3 working: react-faux-dom.
  • Isomorphic charts!
